# ⚽ AI Football Match Prediction

A simple AI-inspired football match prediction web application built using HTML, CSS, and JavaScript.

## 🔍 Features
- Select two football teams
- AI-based probabilistic prediction logic
- Confidence level for predicted winner
- Fully responsive design
- Works offline

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript

## 🚀 How to Run Locally
1. Download or clone the repository
2. Open `index.html` in any modern browser
